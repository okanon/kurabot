クラちゃそBOTの機能一覧

beta 7

    Lv01 権限

  1. mid >> ユーザー識別IDの表示
  2. gid >> グループ識別IDの表示
  3. profile >> ユーザーのプロフィール情報の表示
  4. group >> グループの情報を表示
  5. license >> BOTのライセンスの表示
  6. help >> BOTのヘルプを表示
  7. author >> BOTの作成者の連絡先を表示

    8. Lv02 権限

  8. how2 >> BOTのHow to Useを表示します *1
  9. cancel >> グループ招待のキャンセル機能 *2
  10. creator >> グループの作成者(一番古いユーザー)の連絡先を表示
  11. protect >> 簡易的なグループ保護(招待に関する保護のみ) *3
  12. unprotect >> 簡易的なグループ保護の解除
  13. groupcall >> 現在のグループ通話の状況を表示
  14. noisy >> BOTを通常トークに戻します(silent時のみ)
  15. silent >> BOTがトークしないようにします
  16. gurl >> グループ招待用のURLを発行します *4

    17. Lv03 権限

  17. prevent >> 招待URLによる参加の設定をオフにします
  18. release >> 招待URLによる参加の設定をオンにします
  19. defense >> グループを防衛します(protectコマンドの上位互換) *5
  20. nondefense >> グループ防衛を解除します
  21. cancel_new >> グループの最新の招待のみをキャンセルします *6

*1 helpより詳細なヘルプを表示します。この場合Lv02以上の権限に対するコマンドの詳細です。 *2 招待されている全員をキャンセルします。 *3 グループ招待のみを保護する機能です。Lv03機能のdefenseが有効かされている場合は使えません。 *4 Lv03機能のpreventが有効な場合は取得できません。 *5 グループ招待・グループ名変更・グループ画像変更・招待URLの設定が許可された場合に保護をします。グループの名前・画像・招待URLの設定は元に戻されます。 *6 既に最新の招待がキャンセルされている場合はキャンセルできません。
